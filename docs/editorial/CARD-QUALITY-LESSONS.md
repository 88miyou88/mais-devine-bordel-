# Enseignements éditoriaux cumulés

## Principe de lecture

- Une carte supprimée fournit un signal négatif explicite, mais sa cause reste à déduire lorsqu’aucun motif n’est fourni.
- Une carte excellente dans l’audit fournit un signal positif explicite.
- Une carte neutre indique seulement qu’elle a été vue sans jugement fort.
- Les règles ne deviennent solides qu’après plusieurs retours convergents.
